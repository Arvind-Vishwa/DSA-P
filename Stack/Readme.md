# STACK
## Follow a `LIFO` ordering

### `STL` 
* `stack<int>s` initialization

### `Operation`
* ***`push()`*** --> used to insert in satck
* ***`pop()`*** - It is used to remove the top element in stack
* ***`top()`*** - It return the top element in stack
*  ***`size()`*** - It return the size of stack
